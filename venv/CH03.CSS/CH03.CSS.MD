# CSS

```buildoutcfg
선택자{속성: 값;} - 스타일(CSS)을 적용할 대상 (Selector)
    ✔ 선택자 : html의 어느부분에 적용될 것 인지
    ✔ 속성: 값 : ex) color :red;
    
ex)
div{
    color: red; 
    margin(요소외부여백): 20px;
}

div : 태그 선택자 
```

## 선택자 -기본
```buildoutcfg
✔ 전체선택자
ex)
    *{
        color: red;
    }    
    * : 모든 요소를 선택

✔ 태그선택자
ex)
    li{
        color: red;
    }
    li : 태그이름이 li인 요소 선택

✔ 클래스선택자
ex)
    -html-
    <div>
        <ul>
            <li>사과</li>
            <li>딸기</li>
            <li class="orange">오렌지</li>
        </ul>
        <div>당근</div>
        <p>토마토</p>
        <span class="orange">오렌지</span>
    </div>
    -css-
    .orange{
        color: red;
    }
    .orange : 클래스 속성값이 orange 인 요소 선택 
✔ 아이디선택자
ex)
    -html-
    <div>
        <ul>
            <li>사과</li>
            <li>딸기</li>
            <li id="orange">오렌지</li>
        </ul>
        <div>당근</div>
        <p>토마토</p>
        <span class="orange">오렌지</span>
    </div>
    -css-
    #orange{
        color: red;
    }
    #orange : 아이디 속성값이 orange 인 요소 선택 
✔
```
