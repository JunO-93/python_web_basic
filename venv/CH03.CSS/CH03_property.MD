# 속성(Properties)
```buildoutcfg
* 박스모델

    - 화면에 보여지는 다양한 요소들은 기본적으로 사각형의 형태를 가지고 있다.
    - 가로, 세로 너비가 존재 (width, height)
    - 여백 (margin, padding)
    - 모서리를 둥글게 깎아낼때 (border-radius)
    
* 글꼴,문자
    
* 배경

    - 요소에 배경을 색상이나 이미지를 넣을 수 있음
    
* 배치

    - 특정한 요소를 원하는 좌표에 가져다 놓을 수 있음
    - x,y 좌표가 필요

* 플렉스(정렬)
    
    - 수직, 수평에 대한 정렬 방식을 제공하는 기능
    - 수직 정렬은 자동으로 되는데, 수평 정렬 같은경우는 플렉스를 통해서 정렬한다.
    
* 전환
    
    - 일종의 애니메이션 처리, 전 상태에서 후 상태로 자연스럽게 변환

* 변환
    
    - 요소가 변화 하는것을 말함, 회전, 위치, 크기
    - 2D, 3D 둘 다 가능 

* 띄움
    
    - 뉴스기사 레이아웃 구조에서 많이 보임
    - 사진 주변으로 글자가 흐를 수 있는 기능 

* 애니메이션

    - 하나의 요소가 여러가지 상태에 대한 중간 과정이 전부 만들어지는 기능
    - 전환 효과를 여러단계에 걸쳐서 작성한 거라고 생각해도 무방

* 그리드

    - 플렉스의 확장판 느낌
    - 수평,수직 모두 다 제어할 수 있다. 
    - 2차원의 레이아웃구조를 작업을 할 때 사용한다.
    - 행축, 열축
    
* 다단

    - 글자를 여러개의 단으로 쪼갤 때 사용
    - 글자가 나눠지는 영역을 만들 때 사용

* 필터

    - 화면에서 보여지는 요소에 필터를 사용하여 쓸 수 있음
    - 흐림, 흑백, 반전 외 다른 필터 존재
```
